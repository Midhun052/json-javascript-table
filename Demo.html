JSON - Javascript Table(jjsTable)

jjsTable is a javascript library which can be used to create html table from JSON data. The table is build using JavaScript,BootStrap and JQuery. As BootStrap is used the table is responsive in nature.
Syntax as below,

jjsTable.createTable('tablePlaceHolder',initialResultCount,sampleJSONObject,tableHeadingMappingArray);

tablePlaceHolder - a html place holder to render the table
initialResultCount - initial result count to be displayed in table.
sampleJSONObject - JSON Object containing data.
tableHeadingMappingArray - A mapping between the table column and the table header to be displayed.
